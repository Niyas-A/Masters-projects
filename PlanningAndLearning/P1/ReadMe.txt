To run the code:
part (a): Known maps
    python part_a.py
This will generate all gifs in the pre-existing gif/known_maps folder and grid images in the pre-existing grid_images/known_maps folder

part (b): Random maps
    python part_b.py
This will generate all gifs in the pre-existing gif/random_maps folder and grid images in the pre-existing grid_images/random_maps folder
